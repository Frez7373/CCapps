# CCapps

A suite of tools designed for seamless network communication and management in ComputerCraft/CC:Tweaked environments.

## Features
* **CloudClient**: Lightweight client for remote access.
* **CloudServer**: Robust server management for your CC infrastructure.
* **Cloudnet**: Network diagnostic and utility tool.
* **PrintHub**: Centralized printing management system.

## Quick Install
You can install all components at once using the automated installer. 

1. Open your CC:Tweaked computer terminal.
2. Run the following command to download the installer:
   `wget run https://raw.githubusercontent.com/Frez7373/CCapps/refs/heads/main/Installer.lua`
3. Follow the on-screen instructions.

## Requirements
* ComputerCraft/CC:Tweaked mod installed.
* HTTP requests enabled in the server configuration (`http_enable = true`).
